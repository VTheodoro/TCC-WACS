Sobre o Projeto

O WACS (Wheelchair Automation Control System) é uma solução inovadora de mobilidade para pessoas com deficiência (PCDs) desenvolvida na Etec de Registro como projeto de TCC do curso de Desenvolvimento de Sistemas.

Nossa missão é promover a inclusão e autonomia através da tecnologia, oferecendo uma cadeira de rodas automatizada controlada via smartphone e um aplicativo colaborativo de mapeamento urbano acessível.

Linha do Tempo do Projeto

  • 2025
    Início do Projeto

    Definição do escopo, pesquisa e planejamento inicial.

  • 2025
    Desenvolvimento

    Implementação do hardware, software e testes internos.

  • 2025
    Testes e Validação

    Testes com usuários PCDs, ajustes finais e entrega do projeto.

Perguntas Frequentes

Como funciona o controle da cadeira?

O controle é feito através de um aplicativo mobile que se comunica com a cadeira via Bluetooth, permitindo movimentos precisos e seguros.

O mapa colaborativo é gratuito?

Sim, o mapa colaborativo é totalmente gratuito e aberto para contribuições de todos os usuários.

Como posso contribuir com o projeto?

Você pode contribuir cadastrando locais acessíveis, avaliando estabelecimentos ou participando do desenvolvimento através do GitHub.

Código Aberto no GitHub

O WACS é um projeto de código aberto, disponível para toda a comunidade. Acreditamos que juntos podemos criar uma solução ainda melhor para mobilidade acessível.
